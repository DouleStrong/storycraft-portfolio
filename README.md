# StoryCraft Studio 作品集

这是一个独立的静态展示仓库，用来呈现 StoryCraft Studio 的产品案例、智能 NPC Demo 和产品附录材料。

## 在线入口

- 作品集首页：https://doulestrong.github.io/storycraft-portfolio/
- 产品案例页：`portfolio.html`
- 智能 NPC Demo：`index.html?demo=1`
- 产品附录页：`appendix.html`
- GitHub Pages 根路径会自动进入作品集页。

## 项目定位

StoryCraft Studio 被包装为“剧情与智能 NPC 对话生产后台”产品案例，重点展示：

- 为什么做这个产品，以及它面向什么内容生产场景。
- 如何把智能 NPC 拆成后台生产、角色单聊、AI Agent、Reviewer 和测试集。
- 如何用 PRD 摘要、测试集样例、竞品分析、迭代记录和 Demo 验收清单证明产品判断。

## 仓库边界

本仓库只包含前端静态展示文件，不包含后端服务、数据库、环境变量或真实模型调用。Demo 是静态展示版本，适合直接通过 GitHub Pages 打开体验。
