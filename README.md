这是一个独立的静态展示仓库，用来呈现 AI 产品经理方向的作品集总览、StoryCraft Studio 产品案例、ResearchFlowAgent 产品案例、静态 Demo 和产品文档材料。

## 项目定位

作品集总览页用于概括个人信息、AI 产品经理能力地图和两个独立项目入口，不再把 StoryCraft Studio 作为唯一主页。

StoryCraft Studio 被整理为“剧情与智能 NPC 对话生产后台”产品案例，重点展示：

- 为什么做这个产品，以及它面向什么内容生产场景。
- 如何把智能 NPC 拆成后台生产、角色单聊、AI Agent、Reviewer 和测试集。
- 如何用 PRD 摘要、测试集样例、竞品分析、迭代记录和 Demo 验收清单证明产品判断。
- 如何用规范 PRD 描述需求范围、功能需求、非功能需求、验收标准和边界说明。

ResearchFlowAgent 被整理为“研究型 AI Agent 工作流产品”案例，重点展示：

- 如何把科研文献调研拆成任务理解、提纲规划、知识检索、人工审批和结构化报告输出。
- 如何用来源回溯、中断恢复、人工审核和 benchmark 评测降低 Agent 产品的不确定性。
- 如何通过完成率、耗时、审批轮次和测试通过率描述 AI 工作流的落地效果。
- 如何用静态 Demo 展示“研究问题输入 - Agent 拆解 - 检索来源 - 人工审批 - 结构化报告 - benchmark 指标”的体验链路。

## 仓库边界

本仓库只包含前端静态展示文件，不包含后端服务、数据库、环境变量或真实模型调用。Demo 是静态展示版本，适合直接通过 GitHub Pages 打开体验。
